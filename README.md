# GAN Sketch Colorization

This project aims to generate realistic color images from sketch or grayscale inputs using a conditional GAN framework. The generator will first be implemented as a pix2pix-style baseline and later upgraded with a ConvNeXt V2-inspired architecture including GRN.

## Planned stages
1. Dataset preparation
2. Baseline pix2pix implementation
3. ConvNeXt V2-based generator
4. Evaluation with FID and SSIM
